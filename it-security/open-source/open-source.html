Open Source:

Der Begriff Open Source ("Offene Quelle") bedeutet, dass
der gesamte Quelltext eines Programmes öffentlich zugänglich ist, und jeder den Code lesen kann.
Jeder, der sich mit der Entsprechenden Programmiersprache auskennt, kann auf
diese Weise sehen, was genau das Programm tut. Auch kann er meistens selbst Fehler beheben und sich von dem Programm vielleicht für
eigene Projekte inspirieren lassen.
Der Code kann meistens auf Seiten wie GitHub oder Sourceforge eingesehen werden.
Auch der Code dieser Webseite kann unter https://github.com/safe-im-web/safe-im-web.github.io angesehen werden.
